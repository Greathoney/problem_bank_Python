# 십진수를 이진수로!
이 문제는 파이썬이라면 쉽게 풀 수 있죠. 같이 풀어요. 방법은 다양하지만 저는 이렇게 만들어 볼 겁니다.


## 1. 파이썬의 bin() 함수를 이용해보자.
그러면 문자열 타입의 '0b_____'라고 작성될 것입니다.

## 2. 0b를 떼자
lstrip을 쓰면 됩니다.

이를 모두 합치면
``` python
print('2진수를 모두 뽑아내는 프로그램')
num=int(input('몇 자리의 모든 이진수를 출력할까요?'))
for a in range(2**num,2**(num+1)):
    b=bin(a)
    b=b.replace('0b1','')
    print(b)
```
